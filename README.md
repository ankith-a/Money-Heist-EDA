## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on the **Money Heist (La Casa de Papel)** dataset using Python and popular data science libraries. The goal is to explore the dataset, understand episode-level trends, analyze ratings, identify patterns, and extract meaningful insights through statistical analysis and visualizations.

EDA plays a crucial role in understanding data before building predictive models, and this project demonstrates a complete data analysis workflow from raw data inspection to insight generation.

---

## 🎯 Project Objectives

* Analyze the Money Heist dataset and understand its structure.
* Perform data cleaning and preprocessing.
* Identify missing values and data quality issues.
* Explore rating distributions and episode trends.
* Generate visualizations to uncover hidden patterns.
* Extract meaningful insights using statistical techniques.

## 🔄 Data Analysis Workflow
