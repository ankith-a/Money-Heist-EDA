# 🎭 Money Heist Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Money Heist dataset to identify trends, patterns, and insights related to episodes, ratings, and audience reception.

The analysis includes:

- Data Loading
- Data Cleaning
- Missing Value Analysis
- Statistical Analysis
- Data Visualization
- Rating Analysis
- Exploratory Data Analysis (EDA)

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains information related to the Money Heist series including ratings and episode details.

---

## 📊 Analysis Performed

### Data Exploration

- Dataset Information
- Shape Analysis
- Column Inspection
- Missing Values Check

### Statistical Analysis

- Summary Statistics
- Maximum and Minimum Ratings
- Distribution Analysis

### Data Visualization

- Histogram
- Box Plot
- Rating Distribution
- Exploratory Charts

---

## 📈 Key Insights

- Episode rating distributions were analyzed.
- Highest-rated episodes were identified.
- Outliers in ratings were detected using boxplots.
- Statistical summaries were generated for numerical features.
